# Components Showcase Example

Gallery of all shadcn components demonstrating:
- Tabbed navigation between categories
- Interactive component demos
- All variants and sizes
- Like a "Storybook" for the terminal

## Run
```bash
go run ./cmd/lumina examples/components-showcase/main.lua
```

## Keys
- `1-5` — Switch category tabs
- `j/k` — Scroll within category
- `q` — Quit
